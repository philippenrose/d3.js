Scatter plot of Healthcare vs. Poverty

The above is a scatter plot of Healthcare vs. Poverty. The scatter plots are created with the d3 techniques used in class. They represent each state with circle elements. The graphic is coded in the app.js file of the homework directory. The data is pulled from the data.csv by using the d3.csv function. The scatter plot was designed to be similar to the image in the example

The data set included with the assignment is based on 2014 ACS 1-year estimates: https://factfinder.census.gov/faces/nav/jsf/pages/searchresults.xhtml, but you are free to investigate a different data set. The current data set incldes data on rates of income, obesity, poverty, etc. by state. MOE stands for "margin of error."
